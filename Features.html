Features
Astronomy Picture of the Day (APOD)
  • Feature: Provides a daily image or video with an explanation.
  • API Endpoint: /planetary/apod
  • Data Provided: Title, explanation, date, URL of the image or video.
Mars Rover Photos
  • Feature: Access photos taken by Mars rovers (Curiosity, Opportunity, Spirit).
  • API Endpoint: /mars-photos/api/v1/rovers/{rover_name}/photos
  • Data Provided: Rover name, camera details, image URLs, earth date, and Martian sol.
Earth Observatory Natural Event Tracker (EONET)
  • Feature: Provides information on natural events observed by NASA.
  • API Endpoint: /eonet/v2.1/events
  • Data Provided: Event title, description, categories, geometries, sources, and links.
Near Earth Object Web Service (NeoWs)
  • Feature: Information about asteroids and comets that are close to Earth.
  • API Endpoint: /neo/rest/v1/neo/browse
  • Data Provided: Object name, estimated diameter, close approach data, orbit details.
Exoplanet Archive
  • Feature: Data on exoplanets and their host stars.
  • API Endpoint: /exoplanet/
  • Data Provided: Planet name, host star details, discovery method, and orbital parameters.
NASA Image and Video Library

  • API Endpoint: /search
  • Data Provided: Media type, title, description, date created, keywords, media URLs.
International Space Station (ISS) Current Location
  • Feature: Get the real-time location of the ISS.
  • API Endpoint: /iss-now.json
  • Data Provided: Timestamp, latitude, longitude, message.
Solar System Dynamics (SSD)
  • Feature: Data on planetary positions, orbits, and physical characteristics.
  • API Endpoint: /ssd/api/v1
  • Data Provided: Object name, orbital elements, physical parameters.
Insight: Mars Weather Service API
  • Feature: Provides the latest weather data from the Mars Insight lander.
  • API Endpoint: /insight_weather/
  • Data Provided: Sol (Martian day), temperature, wind speed, atmospheric pressure.
EPIC (Earth Polychromatic Imaging Camera)
  • Feature: Provides images of the Earth captured by the EPIC camera on the DSCOVR spacecraft.
  • API Endpoint: /EPIC/api/natural/images
  • Data Provided: Image date, caption, image URL.
Image Metadata
  • Feature: Retrieve metadata for a specific image from the NASA Image and Video Library.
  • API Endpoint: /metadata/{image_id}
  • Data Provided: Title, description, center, photographer, keywords, location, date created.
